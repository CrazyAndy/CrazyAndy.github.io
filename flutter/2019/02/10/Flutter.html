Slivers

Slivers

Sliver 英文是裂片，条子，口子的意思，从一大块里面切开或裂开一个口子，这个大块指的就是我们应用界面上的ViewPort（可视口），就是显示东西的地方，sliver在这里就是一小块可视口的意思，或者叫可视口的一小块。一般我们会把一些sliver小部件放在CustomScrollView 的slivers属性里面。

我们会在这个slivers 里面放置一些sliver类型的小部件，比如：SliverList，SliverFixedExtentList，SliverGrid，

SliverPadding，SliverAppBar，ScrollNotification，IndexedSemantics。我们可以从CustomScrollView类注释中看到如上描述 。

SliverGrid：网格视图

SliverGrid和之前的GridView类似，一个GridView相当于一个CustomScrollView的sliver属性里面加了一个SliverGrid，如果觉得GridView不能满足需求，可以用一下SliverGrid。查看SliverGrid类定义，我们可以看到它也有count和extent两种方式，构建SliverGrid我们需要提供两个参数，gridDelegate和delegate，前者用来构建网格视图相关的东西，后者主要是构建网格视图里面的项目用的。

import 'package:flutter/material.dart';
import '../model/post.dart';

class SliverDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: CustomScrollView(
        slivers: <Widget>[
          SliverGrid(
            gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
                //主要是可以设置交叉轴上的网格数量
                crossAxisCount: 2,
                crossAxisSpacing: 8.0,
                mainAxisSpacing: 8.0,
                childAspectRatio: 1.0,//显示比例
                ),
            delegate: SliverChildBuilderDelegate(
              (BuildContext context, int index) {
                return Container(
                  child:
                      Image.network(posts[index].imageUrl, fit: BoxFit.cover),
                );
              },
              childCount: posts.length, //网格视图项目数量),
            ),
          ),
        ],
      ),
    );
  }
}

SliverPadding（内边距）与 SliverSafeArea（安全区）

SliverGrid没有padding属性，想要添加内边距，需要去用一个SliverPadding 的小部件。安全区值指的是苹果10以上抬头不能显示全部，只能显示时间，电量，信号，下面显示全部的地方才叫做安全区。

import 'package:flutter/material.dart';
import '../model/post.dart';

class SliverDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: CustomScrollView(
        slivers: <Widget>[
          // SliverPadding(
          //   padding: EdgeInsets.all(8.0),
          //   sliver: SliverGridDemo(),
          // ),
          SliverSafeArea(//这样就放在了苹果安全区内
            sliver: SliverPadding(
              padding: EdgeInsets.all(8.0),
              sliver: SliverGridDemo(),
            ),
          )
        ],
      ),
    );
  }
}

class SliverGridDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return SliverGrid(
      gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
        //主要是可以设置交叉轴上的网格数量

        mainAxisSpacing: 8.0,
        childAspectRatio: 1.0, //显示比例
      ),

          );

}

SliverList：列表视图

我们将上面的SliverGridDemo()替换成SliverListDemo()

class SliverListDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return SliverList(
      delegate: SliverChildBuilderDelegate(
        (BuildContext context, int index) {
          return Padding(
            padding: EdgeInsets.only(bottom: 32.0),
            child: Material(
              borderRadius: BorderRadius.circular(12.0), //圆角效果
              elevation: 14.0, //阴影效果
              shadowColor: Colors.grey.withOpacity(0.5), //阴影颜色
              child: Stack(
                children: <Widget>[
                  AspectRatio(
                    //设置特定比例
                    aspectRatio: 16 / 9, //设置比例
                    child:
                        Image.network(posts[index].imageUrl, fit: BoxFit.cover),
                  ),
                  Positioned(
                    top: 32.0,
                    left: 32.0,
                    child: Column(
                      crossAxisAlignment: CrossAxisAlignment.start,
                      children: <Widget>[
                        Text(
                          posts[index].title,
                          style: TextStyle(fontSize: 20.0, color: Colors.white),
                        ),
                        Text(
                          posts[index].author,
                          style: TextStyle(fontSize: 13.0, color: Colors.white),
                        ),
                      ],
                    ),
                  )
                ],
              ),
            ),
          );
        },
        childCount: posts.length, //网格视图项目数量),
      ),
    );
  }
}
