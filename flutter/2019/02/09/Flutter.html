PageView：页面视图的属性

1，按住屏幕然后左右拖动，如果不到一半的话，就会缩回去，否则翻过去，这个效果叫pageSnapping，默认是true；

2，属性reverse 表示倒置;

3，scrollDirection 滚动方向，默认是横向;

4，滚动回调，onPageChanged 接受 变化后页面的索引值作为参数，索引值从0开始

PageView：用PageController调整页面的显示

PageController是控制pageview小部件显示页面的对象

class ViewDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return PageView(
      // pageSnapping: false,//滚动效果设置
      // reverse: true,//排序默认正向
      scrollDirection: Axis.vertical,//方向
      onPageChanged: (currentPage) => debugPrint('Page:$currentPage'),
      controller: PageController(
        initialPage: 1,//默认当前显示值
        keepPage: false,//默认pageview会记住用户滚动到哪个页面
        viewportFraction: 0.65,//页面占用可视口比例,默认是1
      ),
      children: <Widget>[
        Container(
          color: Colors.brown[900],
          alignment: Alignment(0, 0),
          child: Text('ONE',
              style: TextStyle(fontSize: 32.0, color: Colors.white)),
        ),
        Container(
          color: Colors.grey[900],
          alignment: Alignment(0, 0),
          child: Text('TWO',
              style: TextStyle(fontSize: 32.0, color: Colors.white)),
        ),
        Container(
          color: Colors.blueGrey[900],
          alignment: Alignment(0, 0),
          child: Text('THREE',
              style: TextStyle(fontSize: 32.0, color: Colors.white)),
        ),
      ],
    );
  }
}

PageView.builder按需生成页面

class ViewDemo extends StatelessWidget {
  Widget _pageItemBuilder(BuildContext context, int index) {
    return Stack(
      children: <Widget>[
        SizedBox.expand(
          //expand会占满整个空间
          child: Image.network(
            posts[index].imageUrl,
            fit: BoxFit.cover,
          ),
        ),
        Positioned(
          bottom: 8.0,
          left: 8.0,
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: <Widget>[
              Text(
                posts[index].title,
                style: TextStyle(fontWeight: FontWeight.bold),
              ),
              Text(posts[index].author)
            ],
          ),
        )
      ],
    );
  }

  @override
  Widget build(BuildContext context) {
    return PageView.builder(
      itemCount: posts.length,
      itemBuilder: _pageItemBuilder,
    );
  }
}

网格视图

GridView：网格视图(GridView.count)

class ViewDemo extends StatelessWidget {
  List<Widget> _buildTiles(int length) {
    return List.generate(length, (int index) {
      return Container(
        color: Colors.grey[300],
        alignment: Alignment(0.0, 0.0),
        child:
            Text('Item $index', style: TextStyle(fontSize: 18.0, color: Colors.grey)),
      );
    });
  }

  @override
  Widget build(BuildContext context) {
    return GridView.count(
      crossAxisCount: 3, //交叉轴显示的数量
      crossAxisSpacing: 16.0, //交叉轴方向边距
      mainAxisSpacing: 16.0, //主轴方向边距
      scrollDirection: Axis.horizontal,//滚动方向，默认为纵向
      children: _buildTiles(100),
    );
  }
}

GridView.extent

GridView.extent和GridView.count差不多，唯一的区别就是extent是你需要自定义交叉轴方向的单个title的尺寸，而count是根据你count值进行交叉轴方向 布局的分配。
