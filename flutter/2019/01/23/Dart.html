hideo 筛掉某几个包 show 只使用某几个包

延迟载入

import './phone.dart' deferred as pp;

void main() async {
  var run = true;
  if (run) {
    await pp.loadLibrary();
    var xm = pp.Phone('android');
    xm.startup();
    xm.shutdown();
  }
}

loadLibrary() 方式在需要的时候载入包 可提高程序启动速度 用在不常使用的功能 用在载入时间过长的包
