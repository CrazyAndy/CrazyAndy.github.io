Dart基础学习——变量-001

一、弱类型

var

　　如果没有初始值，可以变成任何类型

　　var a;

　　a = ‘yo yo qie ke nao’;

　　a = 123;

　　a = true;

　　a = {‘key’:’xiha’};

　　a = [‘eee’];

Object

　　动态任意类型，编译阶段检查类型。

　　Object a = ‘yoyo’;

　　a = 123;

　　a = [22];

　　a.p();

dynamic

　　动态任意类型，编译阶段不检查类型。

　　dynamic a = ‘xiha’;

　　a = 123;

　　a.b();

比较var , Object , dynamic

　　唯一区别是var 如果有初始值，类型被锁定。

二、强类型

声明类型

声明类型之后，类型被锁定

String a;

a = ‘ducalcat’;

a = 123;//这样会导致错误

常见类型

名称 说明
num 数字
int 整型
double 浮点
bool 布尔
String 字符串
StringBuffer 字符串 buffer
DateTime 时间日期
Duration 时间区间
List 列表
Sets 无重复队列
Maps kv 容器
enum 枚举

默认值

一切都是Object，默认为null

如何使用

  • 在写api接口的时候，请用 强类型 ，一旦不符合约定，接受数据时能方便排查故障
  • 在写小工具时，可以用 弱类型 ，这样代码写起来比较快，类型自动适应
