linux搭建httpd

一、实现原理

　　httpd是Apache超文本传输协议(HTTP)服务器的主程序。它被设计为一个独立运行的后台进程，它会建立一个处理请求的子进程或线程的池对外提供服务。我在安装ambari的时候需要先搭建httpd服务，并将ambari的安装包上传到httpd的文档目录下以便安装,如果使用ambari在线安装，它有个半小时限制，超过半小时安装不完就停止，而ambari安装包有6G多。我们可以使用httpd服务对外提供文件下载，在linux上面搭建还是比较简单的。

二、准备

　　1. 有些版本的linux已经安装httpd服务，所以我们先查看httpd是否已安装

# a. 查看是否有httpd进程正在运行
ps -ef|grep httpd
# b. 查看httpd是否已经安装
rpm -qa httpd
# c. 查看linux系统服务中有没有httpd
chkconfig --list

三、安装httpd

　　1. 安装及验证httpd

# yum -y install httpd
# rpm -qa |grep httpd

　　2. 使用rpm方式寻找配置文件

# rpm -qc httpd

　　3. 常用的配置文件

/etc/httpd/conf.d/*.conf        ###辅助配置文件
    /etc/httpd/conf/httpd.conf      ###主配置文件
    /etc/sysconfig/httpd            ###httpd工作模式配置文件

　　4. 启动脚本：

　　/etc/rc.d/init.d/httpd

　　5. 日志文件目录:

　　 /var/log/httpd
        access_log：访问日志
        error_log: 错误日志

　　6. 站点文档目录：（站点根目录） 　　/var/www/html ,这个，还有端口都可以在 /etc/httpd/conf/httpd.conf 配置文件里面配置，端口默认是80
